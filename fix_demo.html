Technical Details

  • Root Cause: The modifySvg function only processed path, g, and rect elements
  • Missing Elements: circle and ellipse elements with fill="currentColor" were ignored
  • Solution: Added support for circle and ellipse elements with proper CSS class generation
  • CSS Classes:
    • circle[fill="currentColor"] → .c{n}-fill{fill:currentColor}
    • circle[stroke="currentColor"] → .c{n}-stroke{stroke:currentColor}
    • ellipse[fill="currentColor"] → .e{n}-fill{fill:currentColor}
    • ellipse[stroke="currentColor"] → .e{n}-stroke{stroke:currentColor}
