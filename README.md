# Icon Collection

This repository contains a collection of icon packages and icon parsers.

## How to use

```bash
npm install
# or
yarn install
```

* Get svg icons with react-icons

  ```bash
  node parsers/react-icons.js
  ```

* Get mxlibrary file

    ```bash
    node parsers/svg2mx.js react-icons
    ```
